使用方法

引入文件

<script src="js/jquery.min.js"></script>
<script src="js/jquery.scroll-follow.js"></script>

CSS

必须为滚动的元素指定相对定位，否则无法滚动。

#example {
    position : relative;
}

调用方法

Scroll Follow 的使用非常简单，直接为某个元素调用方法即可。

$(function() {
    $.scrollUp();
});

参数

参数 说明
speed 滚动速度，单位为 ms，数值越小，速度越快，默认为 500
easing 如果你使用了 easing，可以使用 easing 的参数
offset 偏移，相对于顶部或底部的偏移（取决于 relativeTo 参数），单位为 px，默认为 0
container 滚动元素的容器（直接的父元素）的 id
killSwitch 开关 id，默认为 killSwitch，需要用到 jQuery Cookie 插件，以保留元素的状态
onText 滚动状态时开关文字，默认为 Turn Slide Off
offText 停止滚动状态时开关文字，默认为 Turn Slide On
relativeTo 相对与顶部（top）或底部（bottom）滚动，默认为顶部（top）
delay 延迟滚动，单位为毫秒 ms，默认为 0
